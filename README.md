# Data Structures

This repository contains instructional materials for an undergraduate **Data Structures** course, including lectures, labs, and course documentation.

## Repository Contents

- **Syllabus/**  
  Course policies, schedule, learning objectives, and assessment overview.

- **Labs/**  
  Programming labs and guided exercises designed to reinforce core data structures and algorithms.

- **Presentations/**  
  Lecture slides and supporting instructional materials.

- **Exams/**  
  Assessment materials used in the course.  
  *(Note: Solutions are intentionally not included.)*

## Topics Covered

- Abstract Data Types (ADTs)
- Algorithmic complexity and Big-O notation
- Arrays and linked lists
- Stacks and queues
- Trees and binary search trees
- Heaps and priority queues
- Hash tables
- Graphs and graph traversal
- Recursion and divide-and-conquer techniques
